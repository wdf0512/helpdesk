# Tech Stack

## Frontend
- **React** (Vite) — UI framework
- **React Router** — client-side routing
- **Tailwind CSS** — styling
- **shadcn/ui** — component library

## Backend
- **Node.js + Express** — API server

## Database & ORM
- **PostgreSQL** — relational database
- **Prisma** — ORM

## Auth
- **Database sessions** — server-side session storage backed by PostgreSQL

## AI
- **Claude API (Anthropic)** — ticket classification, summaries, suggested replies, auto-responses

## Email
- **Resend** — outbound email
- **Mailgun** — inbound email (webhook-based ticket creation)

## Deployment
- **Docker** — containerisation for all services
